Step-by-Step Execution — Start Here
Pick your path: Option A (5 min, no API key) · Option B (Groq free tier) · Option C (full Docker stack)
OPTION A — Fast simulated demo (no API key needed)
1
Check you're in the NeuroCI directory
PowerShell:
terminal
cd c:\Users\mahad\OneDrive\Desktop\mahi\DevSecOps
2
Check Python is installed
terminal
python --version
Output should show Python 3.9+
3
Activate virtual environment
terminal
.\.venv\Scripts\Activate.ps1
Prompt should show (.venv) prefix
OPTION C — Full Docker system (all 7 services, real LLM)
1
Verify Docker is running
terminal
docker ps
Should show docker is responsive (even if no containers yet)
2
Configure .env file
terminal
cat .env.example
Edit `.env` and set:
  • GITHUB_WEBHOOK_SECRET=demo-secret-123
  • GITHUB_TOKEN=any-placeholder
  • LLM_PROVIDER=groq (or gemini or ollama)
  • GROQ_API_KEY=gsk_your_key (if using Groq)
3
Start all 7 Docker services (first run: 3-5 min)
terminal
docker compose up -d
Wait for pull to complete. Then verify:
4
Verify all 7 services are running
terminal
docker compose ps
All 7 should show Status: Up:
  • webhook (FastAPI)
  • worker (Celery)
  • redis
  • chromadb
  • opa
  • prometheus
  • grafana
5
Test API health endpoint
terminal
curl http://localhost:8000/health
Should return HTTP 200 with {"status":"ok"}
6
Fire a live test webhook in a NEW terminal window
Keep first terminal showing worker logs:
terminal — window 1 (keep open)
docker compose logs -f worker
In a second terminal window, send the webhook:
terminal — window 2
python scripts/send_signed_webhook.py --payload .\tests\fixtures\sample_logs\sample_failure.json
Watch window 1 carefully. You will see all 11 steps fire in real-time:
  • webhook.received
  • signature verified
  • logs fetched
  • failure classified
  • RAG lookup (ChromaDB)
  • patch generated
  • patch validated
  • policy evaluated
  • PR created
7
View Grafana dashboard (optional)
Open http://localhost:3000 → Username: admin, Password: admin
You will see live MTTR and fix rate metrics updating in real time.
8
Clean up when done
terminal
docker compose down
Quick reference URLs:
API docs: http://localhost:8000/docs
Prometheus: http://localhost:9090
Grafana: http://localhost:3000
ChromaDB: http://localhost:8001
